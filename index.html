🎨 CSSスニペット

追加CSSに貼るだけで使えるスタイル集。記事ごとにクラスを付けて使います

ネストリスト：中抜き丸
子リストの箇条書きを同じピンク色の中抜き丸に変更する
ul.scoped-nested > li::before { background: transparent !important; border: 2px solid #F24BE1 !important; box-sizing: border-box !important; }
📋 使い方
  1. 上の CSS を 外観 → カスタマイズ → 追加CSS に貼り付けて保存
  2. 記事のコードエディターで子リストの <ul> を開く
  3. class="wp-block-list" を class="wp-block-list scoped-nested" に変更して保存
<ul class="wp-block-list scoped-nested"> <li>サブ項目1</li> <li>サブ項目2</li> </ul>
